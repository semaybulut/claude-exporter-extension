# Claude Exporter Extension
English description below.

Herhangi bir Claude sohbetini tek tıkla temiz bir Markdown dosyası olarak indir.

## Özellikler

- Her Claude sohbetinde tek tıkla export
- Zaman damgalı, temiz Markdown çıktısı
- Python yok, terminal yok, kurulum yok
- Tamamen tarayıcında çalışır — hiçbir veri dışarı çıkmaz

## Kurulum

1. [Releases](../../releases/latest) sayfasından ZIP'i indir
2. ZIP'i aç
3. Chrome'u aç → `chrome://extensions/` adresine git
4. Sağ üstte **Geliştirici modu**'nu aç
5. **"Paketlenmemiş öğe yükle"** → klasörü seç
6. [claude.ai](https://claude.ai) üzerinde herhangi bir sohbet aç
7. Sağ alttaki **"⬇️ Export MD"** butonuna tıkla

## Kullanım

1. claude.ai'da herhangi bir sohbet aç
2. Sağ alttaki **"⬇️ Export MD"** butonuna tıkla
3. `.md` dosyası otomatik olarak indirilir

## Gereksinimler

- Google Chrome (veya Chromium tabanlı tarayıcı: Brave, Edge, Arc)
- Chrome extensions'ta Geliştirici modu açık olmalı

# Claude Exporter Extension

Export any Claude conversation as a clean Markdown file — directly from your browser.

## Features

- One-click export from any Claude conversation
- Clean Markdown output with timestamps
- No Python, no terminal, no setup
- Works entirely in your browser — no data leaves your machine

## Installation

1. Download the latest ZIP from [Releases](../../releases/latest)
2. Unzip the folder
3. Open Chrome → `chrome://extensions/`
4. Enable **Developer mode** (top right toggle)
5. Click **"Load unpacked"** → select the unzipped folder
6. Open any conversation on [claude.ai](https://claude.ai)
7. Click **"⬇️ Export MD"** button (bottom right)

## Usage

1. Open any conversation on claude.ai
2. Click the **"⬇️ Export MD"** button at the bottom right
3. A `.md` file is downloaded automatically

## Requirements

- Google Chrome (or any Chromium-based browser: Brave, Edge, Arc)
- Developer mode enabled in Chrome extensions
